# Bootstrap 模态框 
--- 

<!--break-->  


判断页面某个模态框是否显示或隐藏：
``` 
var modal_status = 0;//默认0， 0表示隐藏
  
$('#myModal').on('show.bs.modal', function (e) {
`modal_status = 1; // 1表示显示
})
  
$('#myModal').on('hidden.bs.modal', function (e) {
`modal_status = 0;
})
```
